Критерии готовности для admin.py

- Каждая модель зарегистрирована через декоратор admin.register().
- Для каждой модели создан отдельный admin-класс.
- Для каждой модели настроены list_display, list_filter, search_fields, list_editable и list_per_page.
- В ТЗ указаны поля для отображения, фильтрации, поиска и редактирования по каждой модели.
- Админ панель открывается без ошибок.
